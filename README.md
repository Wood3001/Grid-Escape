# Grid-Escape

Grid-Escape is a puzzle game that challenges players to escape a series of interconnected grids by solving puzzles and navigating through obstacles. 

## Technologies Used

Grid-Escape is built using the following technologies:

- HTML5
- CSS3
- JavaScript
- jQuery
- GSAP Draggable

## Installation

To run Grid-Escape locally on your machine, follow these steps:

1. Clone this repository to your local machine.
2. Open the `index.html` file in your preferred web browser.

## Usage

To play Grid-Escape, follow these instructions:

1. Use your mouse to drag the blue blocks.
2. Clear a path for the escapee (orange block).
3. Navigate through obstacles to reach the exit portal.
